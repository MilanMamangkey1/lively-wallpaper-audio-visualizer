# Media Parallax - Audio Visualizer

An interactive audio visualizer wallpaper for [Lively Wallpaper](https://github.com/rocksdanister/lively) that combines music visualization, album art, and mouse parallax effects.

## Features

- **Audio Visualization**: Real-time audio spectrum analyzer with customizable dot-based visualizer
- **Album Art Display**: Shows current playing media album art with dynamic background
- **Mouse Parallax**: Interactive background that responds to mouse movement
- **Color Extraction**: Automatically extracts colors from album art for dynamic theming
- **Customizable Settings**: Adjustable amplitude, dot size, and spectrum modes

## Configuration

Access wallpaper settings through Lively Wallpaper to customize:

- **Parallax**: Enable/disable mouse parallax effect
- **Amplitude**: Adjust visualizer height (0-50)
- **Dot Size**: Control visualizer dot size (0-5)
- **Spectrum Mode**: Choose from:
  - Concentrated
  - Center-split
  - Keep original

## Usage

The wallpaper will automatically:
- Detect system audio and display the visualizer
- Show album art and track information when media is playing
- Update background colors based on album art
- Respond to mouse movement with parallax effect (if enabled)
